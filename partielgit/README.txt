Invite de commandes

Je me place dans le dossier de mon projet :

cd C:\Users\gauth\OneDrive\Bureau\partielgit

Je crée mon répertoire :

git init

Je l'ajoute sur Github dans le repository crée au préalable nommé partielgit :

git remote add origin https://github.com/gauthierm06/partielgit.git

Ajoutez un fichier code.txt avec « hello world » et faites un commit des changements:

git add code.txt

git commit code.txt

Mon éditeur visual studio code s'ouvre et j'entre le message lié a mon premier commit.

Je modifie mon fichier texte Hello world en Hello git dans mon répertoire puis j'effectue la commande:

git commit code.txt

Mon éditeur s'ouvre et j'entre le message de la deuxieme modification : remplacement de hello world par hello git.

Je supprime ensuite mon fichier code.txt de mon répertoire puis je fais un git commit.

J'ajoute mon fichier README.txt

git add README.txt

Puis j'effectue un commit afin de le mettre à jour.
